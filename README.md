# The Aurora X1
## Backstory
A while back in August 2025, I had a conversation with my mother about me getting a phone for my 13th birthday.
The response to this was that i was not mature enough for a phone. This spark gave me the idea that maybe, i can make my own phone
## The Result
I ended up fufilling that idea and ended up creating the Aurora X1 Mobile Cellular Device Developer Kit
## What is in this repository?
This repo contains the following
1. CAD Designs
2. Parts Lists (X-BOM.CSV)
3. Documentation on development

## Compute Power
Personally i used an RPI-5 with 16GB of RAM. That probably is a bit excessive for a normal use case.
Display components were from adafruit and are listed in the parts list
